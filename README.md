It`s test relise client web app for https://github.com/SoVa-web/app_mockservers

To start the graphical interface in the ui_simple_mockservices project directory, you need to execute the "npm run simple_start_ui" command.

First, in the simple-mock-services (app_mockservers on GitHub) project directory, execute the command "npm run simple_start_server"

App uses ports 3001, 5002, 5003 for local software deployment and 6379 for Redis server.
